# 一个简单的koa小demo

```start with "$ node app.js" ```

项目默认启动在3000端口下，可在app.js中的```app.listen()```中更改。

使用node + koa2搭建了本地服务，加上vue + vue-router做了简单的接口测试与本地登录验证。
